# Playwright MCP 스크린샷 갤러리 프롬프트

Playwright MCP를 사용해서 아래 웹사이트들의 스크린샷을 찍고, 저장한 이미지를 갤러리 뷰로 만들어주세요.

대상 URL: 
```text
https://learn.chatgpt.com/docs
```

진행 순서:

1. 아직 코드를 수정하지 말고 캡처 계획을 먼저 작성해주세요.
2. 계획에는 URL, viewport, 저장 파일명, 실패 시 대체 방법을 포함해주세요.
3. 제가 승인하면 Playwright MCP로 각 URL을 방문하고 스크린샷을 저장해주세요.
4. 저장한 이미지를 사용해서 정적 갤러리 뷰를 구현해주세요.
5. 구현 후 브라우저에서 갤러리를 다시 확인해주세요.

캡처 조건:

- 데스크톱 viewport는 1365x768로 사용합니다.
- 스크린샷은 `public/site-shots/`에 저장합니다.
- 페이지가 로드될 때까지 기다린 뒤 캡처합니다.
- 로그인이나 개인 정보가 필요한 페이지는 사용하지 않습니다.
- 쿠키 배너나 팝업은 닫을 수 있는 경우에만 닫습니다.

갤러리 구현 조건:

- `index.html`, `styles.css`, `gallery.js` 정도의 단순 구조를 사용합니다.
- 새 라이브러리는 설치하지 않습니다.
- 카드에는 사이트 이름, URL, 캡처 날짜, 한 줄 메모를 표시합니다.
- 이미지는 16:9 비율로 보이게 합니다.
- 모바일에서는 1열, 데스크톱에서는 3열 그리드로 보이게 합니다.

검증 리포트 형식:

- Chrome devtools mcp를 사용해서 콘솔 에러가 발생하는지, 런타임 에러 발생하는지 점검합니다.
```text
## 스크린샷 갤러리 검증 리포트

- 확인한 URL:
- 캡처한 사이트:
- 생성된 이미지 파일:
- 갤러리에서 확인한 항목:
- 레이아웃 확인:
- 콘솔 에러:
- 실패/대체 처리:
- 사람이 직접 확인할 부분:
```
