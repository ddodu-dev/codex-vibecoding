# Chrome DevTools MCP 디버깅 프롬프트

Chrome DevTools MCP를 사용해서 아래 로컬 페이지를 확인해주세요.

```text
http://localhost:4174
```

조건:

- 아직 코드는 수정하지 마세요.
- 페이지가 열리는지 확인해주세요.
- 화면에서 비어 있거나 이상한 부분을 관찰해주세요.
- 콘솔 에러 메시지를 확인해주세요.
- 에러가 발생한 파일과 줄을 찾아주세요.
- 에러의 직접 원인 후보를 정리해주세요.
- 추측과 브라우저에서 확인한 근거를 나눠주세요.

리포트 형식:

```text
## 디버깅 리포트

- 확인한 URL:
- 화면 증상:
- 콘솔 에러:
- 발생 파일/줄:
- 브라우저에서 확인한 근거:
- 직접 원인 후보:
- 먼저 확인할 코드:
- 아직 수정하지 않은 이유:
```
