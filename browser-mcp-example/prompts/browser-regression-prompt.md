# 수정 후 재검증 프롬프트

방금 수정한 런타임 에러를 같은 브라우저 흐름으로 다시 검증해주세요.

조건:

- Chrome DevTools MCP로 `http://localhost:4174`를 다시 확인해주세요.
- 새 기능을 추가하지 마세요.
- 처음 실패했던 항목을 중심으로 확인해주세요.
- 추천 미션 카드가 채워지는지 확인해주세요.
- 미션 목록 3개가 렌더링되는지 확인해주세요.
- 콘솔 에러가 새로 생겼는지 확인해주세요.
- 수정 전/후 차이를 구분해주세요.

리포트 형식:

```text
## 수정 후 재검증 리포트

- 수정 전 실패:
- 수정한 파일:
- 수정한 내용:
- 다시 확인한 동작:
- 수정 후 결과:
- 콘솔 에러:
- 새로 발견한 문제:
- 남은 위험:
```
